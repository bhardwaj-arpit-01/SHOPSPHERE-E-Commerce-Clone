# ShopSphere

ShopSphere is a responsive e-commerce web application inspired by modern online marketplaces. The project is built to practice and demonstrate front-end development skills using HTML, CSS, and JavaScript.

## Features
- Responsive design
- Product catalog
- Product search
- Shopping cart functionality
- Category filtering
- Local storage persistence
- Modern UI/UX

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)

## Purpose
This project was created to strengthen JavaScript fundamentals and gain hands-on experience building real-world e-commerce interfaces.
